Rules

1. click on roll dice to get random number
2. active player can click roll dice and sum of there rolls is calculated in current.
3. if active player click hold, then current is added to player score.
4. if number is equal to 1 then active player will be switched and current value becomes zero.
5. if any player reaches greater than or equal to zero then that player is winner.
6. To start the new game at any time, click on New Game.

7. Winner player board is turned black
